Python Conditional statements

We can use these syntaxs in any senario condition applies

Syntax for "if" condition

>>>if(condition):
>>>....statements #here "." reprasents indentation we can ignore while writting program

Example:

>>>x=5
>>>if(x<5): #here x<5 condtion is true then it follows next statement other wise it not get into next statement
>>>....print("true")

#Output
>>>true

Syntax for "if/else" condition

>>>if(condition):
>>>....statements #here "." reprasents indentation we can ignore while writting program
>>>else
....statements

Example:

>>>x=5
>>>if(x<5): #here x<5 condtion is true then it follows next statement other wise else statements will work
>>>....print("true")
>>>else
>>>...print("false")

#Output
>>>false
