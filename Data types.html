Built-in Data Types

In programming, data type is an important concept.

Variables can store data of different types, and different types can do different things.
Python has the following data types built-in by default, in these categories:
  • Text Type: str
  • Numeric Types: int, float, complex
  • Sequence Types: list, tuple, set
  • Mapping Type: dict
  • Boolean Type: bool
  • Binary Types: bytes, bytearray, memoryview

>>> x=5 here x is int

>>> x=5.2 here x is float

>>> x=5+j9 here x is complex

>>> x="python" here x is string

>>> x=true here x is boolean

>>> x={"c":1} here x is dict

>>> x=[1,2,3] here x is List

>>> x=(1,2,3) here x is Tuple

>>> x={1,2,3} here x is set

To Check the datatype of given data,We use type function:"type()"
