Python Loops

Generally,Other languages(like in Java,C,C++,etc..) contains three types of loops like for loop,whlie loop,do-while loop

So,we compared to other languages Python contains only two types of loops. We see below

  1. for loop
  2. while loop

For Loop

>>>for i in range(x):
>>>....statements #here "." reprasents indentation we can ignore while writting program

Example:

>>>x=5
>>>for i in range(x):
>>>....print(i)

#Output
0
1
2
3
4

>>>while(condition):

>>>.....statements
